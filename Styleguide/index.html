Process Analysis

This was a projet from a class called Interactive Art and Technology: Internet Computing Technology in which we learned how to build web application using PHP. In this project I integrated a database with PHP which could prefrom queries to your specific selection. The tools used here was PHP, SQl, and a web server using XAMPP. Since this was my first time working with SQL, I had trouble all over using SQL from doing input error check to concentinating SQl strings, and to even setting up a DB using root login. However, I was able to break apart each problem with splitting them into smaller chunks going over each one by one. I solved each problem going over the documentation and teh textbook we were given. In the end, I was able to complete the project with all funtionalities included. Users were able to query parameters using order number and order date. They were able to select on the control which information to be displayed when the query is successful.

Process Analysis 2

This was a final project for the course Interactive Arts and Technology: Interface Design. For this project my group Liv, Zack, and me had to create and design a mobile app from scratch. We began by going to the drawing board brainstorming ideas of what we would like to make, and we decided on a app to teach people how to mix drinks. We then went through an interactive design process first by determining our design domain our target group and began creating personas for people who would use our app then we started to sketch our mock up interfaces. As we had decided on what we wanted we began to make prototypes using tools like Sketch and Photoshop. When everything was to our expectation we created an interactive prototype using Protopie.
